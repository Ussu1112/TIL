#### 다이나믹 프로그래밍이란

- 다이나믹 프로그래밍의 성립 조건
- 부분 반복 문제(Overlapping Subproblem)
  - 반복되는 동일한 연산
  - 큰 문제를 작은 문제로 분할 가능
  - 
- 최적 부분 구조(Optimal Substructure)
  - 문제의 부분에서 구한 답을 바탕으로 전체 문제의 답을 도출
  - 최적의 정답이 존재
